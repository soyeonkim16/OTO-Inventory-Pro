# OTO 재고관리 v1.8

## 변경 내용
- 관리자 전용 거래처 삭제 기능 추가
- 거래처 삭제 전 확인창과 거래처명 재입력
- 거래처 삭제 후에도 기존 출고 이력은 보존
- 상품 재고 화면에서 모델명 열 제거
- 상품 등록·수정 화면에서 모델명 입력 제거
- 상품 검색에서도 모델명 제외
- 기존 데이터베이스 `sku` 열은 호환성을 위해 유지하지만 앱에서는 사용하지 않음

## 업데이트 순서
1. Supabase SQL Editor에서 `SUPABASE_v1_8_거래처삭제기능.sql` 실행
2. v1.5 상품 삭제 SQL을 아직 실행하지 않았다면 함께 실행
3. GitHub에 v1.8 프로젝트 업로드 후 Commit
4. Netlify 자동 배포 확인

## 주의
거래처 삭제 SQL을 실행하지 않으면 앱의 거래처 삭제 버튼에서 오류가 표시됩니다.
