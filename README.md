# OTO Inventory Pro v2.1.0

## 업데이트 내용
- 제공된 새 OTO 이미지로 앱 아이콘 교체
- Android/PWA 홈 화면 아이콘 교체
- 브라우저 탭 favicon 교체
- 로그인 화면 로고 교체
- 상단 헤더 로고 교체
- Apple 홈 화면 아이콘 교체
- 파란색/코랄 브랜드 색상 적용
- 로딩 화면 애니메이션 추가하지 않음
- 전체 글씨 두께를 기존보다 조금 가볍게 조정
- v2.0.2의 상품등록·직원관리·삭제 기능 유지

## 업로드 방법
압축을 풀고 안의 파일과 폴더 전체를 GitHub 저장소 최상단에 덮어쓰세요.

커밋 메시지:
`v2.1 아이콘 및 글씨 두께 업데이트`

Netlify 자동 배포가 완료된 후 브라우저에서 Ctrl+F5로 새로고침하세요.

Android 홈 화면 아이콘이 이전 것으로 남으면 기존 설치 앱을 삭제한 뒤 Chrome에서 다시 앱 설치를 진행하세요.
