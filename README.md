# OTO 재고관리 v1.6

## 추가 기능
- 거래처 카드를 클릭하면 해당 거래처의 출고현황 표시
- 일자별 총 출고수량 표시
- 날짜별 출고 품목과 품목별 수량 표시
- 거래처별 누적 출고수량 표시
- 시작일·종료일 기간 검색
- 거래처 출고내역 CSV 다운로드
- 모바일에서는 거래처 목록 아래에 출고현황 표시

## 기존 기능 유지
- 아이디/비밀번호 로그인
- 자동 로그인
- Pretendard
- 모델명·사이즈·색상 분리
- 관리자 상품 삭제
- 실시간 재고 동기화

## SQL
v1.6 기능은 기존 stock_logs 데이터를 이용하므로 추가 SQL이 필요하지 않습니다.
v1.5 상품 삭제 SQL을 아직 실행하지 않았다면 `SUPABASE_v1_5_상품삭제기능.sql`만 실행하세요.
