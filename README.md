# OTO Inventory Pro v2.2.0

## 업데이트 내용
- 통계 메뉴 및 상품별 누적 출고 통계 화면 삭제
- 재고 목록의 입출고 버튼 삭제
- 입출고 메뉴 상단에 상품 선택 및 입출고 등록 기능 배치
- 입출고 등록 직후 재고 수량과 입출고 내역을 화면에 즉시 반영
- 백그라운드에서 Supabase 데이터를 다시 불러와 최종 동기화
- 기존 실시간 Realtime 구독 유지
- v2.1 아이콘과 가벼운 글씨 두께 유지
- 상품등록, 직원관리, 거래처/상품 삭제 기능 유지

## 업로드 방법
압축을 풀고 안의 파일과 폴더 전체를 GitHub 저장소 최상단에 덮어쓰세요.

커밋 메시지:
`v2.2 입출고 화면 및 실시간 반영 개선`

Netlify 자동 배포 완료 후 브라우저에서 Ctrl+F5를 누르세요.
안드로이드 설치 앱은 완전히 종료한 뒤 다시 실행하면 새 버전이 적용됩니다.
