# OTO Inventory Pro v2.3.1

## 수정 내용
- 연결된 상품이 이미 삭제된 입출고 기록도 삭제 가능
- 상품이 존재하면 기존처럼 재고를 원상복구
- 상품이 없으면 재고는 건드리지 않고 내역만 삭제
- 관리자 전용 삭제 버튼 유지
- 삭제 직후 화면 즉시 반영

## 필수 SQL
Supabase SQL Editor에서 아래 파일을 실행하세요.

`SUPABASE_v2_3_1_삭제보강.sql`

기존 v2.3 함수는 자동으로 교체됩니다.

## GitHub 커밋 메시지
`v2.3.1 삭제된 상품 입출고 내역 삭제 보강`


## v3.0 추가 기능
- 거래처 목록 가로형 테이블 및 정렬
- 거래명세표 A4 가로 인쇄/PDF 저장
- 거래명세표 저장 및 재불러오기(브라우저 기기 저장)
- 제품별 최근 단가 자동 기억


## v3.1 추가 기능
- 상품별 도매가/소매가 저장
- 거래처별 기본 단가 구분(도매/소매)
- 거래명세표에서 도매가/소매가 일괄 전환 및 단가 직접 수정
- A4 세로 1장에 공급받는자 보관용/공급자 보관용 2단 출력
- 가운데 절취선
- 모바일 거래처 카드 목록 표시 오류 보강
