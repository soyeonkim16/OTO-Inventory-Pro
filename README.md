# OTO Inventory Pro v2.3.1

## 수정 내용
- 연결된 상품이 이미 삭제된 입출고 기록도 삭제 가능
- 상품이 존재하면 기존처럼 재고를 원상복구
- 상품이 없으면 재고는 건드리지 않고 내역만 삭제
- 관리자 전용 삭제 버튼 유지
- 삭제 직후 화면 즉시 반영

## 필수 SQL
Supabase SQL Editor에서 아래 파일을 실행하세요.

`SUPABASE_v2_3_1_삭제보강.sql`

기존 v2.3 함수는 자동으로 교체됩니다.
